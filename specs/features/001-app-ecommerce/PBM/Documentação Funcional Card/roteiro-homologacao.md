# Roteiro homologação

**Fonte:** https://developer.funcionalmais.com/docs/gateway-credenciados/roteiro-homologacao

---

Para conclusão da homologação é preciso documentar o que foi desenvoldido da seguinte forma:

Criar um documento em WORD/PDF com os prints do passo a passo de como se realiza uma venda dentro software desenvolvido.
Inclusive se necessário colocar obs. de como chegar as determinadas telas.

Pré-requisto para liberação além do envio da documantação é preciso encaminhar os comprovantes de pelo menos uma vendas (Onde Validaremos o Layout do comprovante do Relatório Gerencial).

### Dados para utilização de vendas teste:

Clientes testes e CRM solicitar ao time de EDI, para o analista responsável pela integração.

Medicamento: Qualquer medicamento, e Data da Receita: Data da simulação.

>
> Para validação da homologação é preciso mostrar todos os cenários especificados de compra:
>
> - Total Benefício Farmácia
> - Parcial
> - à vista

>
